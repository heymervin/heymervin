# Hey, I'm Mervin

**AI Product Engineer** — I build AI-powered products from zero to deployed. CS graduate, classically trained (ABAP/SAP), building toward founding my own company.

I work across the full stack — React, TypeScript, Supabase, Node.js — with deep integrations into LLMs (Claude, Gemini, OpenAI) and automation platforms (GoHighLevel, Make.com). The products I build for clients are prototypes for the ones I'll build for myself.
